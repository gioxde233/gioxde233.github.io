Software Hardware Firmware
Son los programas y el sistema operativo que se utilizan en un dispositivo. El software es más versátil que el firmware y se puede utilizar en varios dispositivos Son las partes físicas de un dispositivo, como la placa base, el procesador, la memoria y la tarjeta gráfica. El hardware es tangible, es decir, se puede tocar y sentir Es un tipo de software que se integra en el hardware específico de un dispositivo. El firmware es intangible, es decir, no tiene forma física

Que es el hardware?

El hardware es el conjunto de componentes físicos que permiten el funcionamiento de un ordenador o computadora:

-Teclado

-Monitor

-Mouse

-Placa base

-CPU

-Ventiladores

El hardware se refiere a los componentes físicos de un sistema informático, es decir, las partes tangibles que permiten que la computadora funcione. Incluye dispositivos como la placa base (motherboard), procesador (CPU), memoria RAM, disco duro, tarjeta gráfica, fuente de alimentación, y periféricos como el teclado, ratón y pantalla. En conjunto, estos elementos permiten ejecutar y controlar el software, proporcionando la infraestructura necesaria para el funcionamiento de los sistemas informáticos.

La placa base (motherboard) conecta todos los elementos, como la CPU (procesador), que ejecuta las instrucciones del sistema y procesa los datos. La memoria RAM almacena temporalmente los datos que utiliza la CPU para ejecutar tareas, mientras que el disco duro (HDD) o la unidad de estado sólido (SSD) ofrecen almacenamiento permanente para el sistema operativo, programas y archivos. La tarjeta gráfica (GPU) es responsable de procesar y mostrar los gráficos en la pantalla, especialmente en tareas que requieren un alto rendimiento visual, como juegos o edición de video. La fuente de alimentación (PSU) convierte la corriente alterna de la red eléctrica en corriente continua para que todos los componentes puedan funcionar correctamente. Además, los periféricos, como el teclado, ratón y pantalla, permiten la interacción del usuario con el sistema. Todos estos componentes trabajan en conjunto para hacer que una computadora o dispositivo realice diversas funciones, desde procesar datos hasta mostrar información visual o interactuar con el usuario.
